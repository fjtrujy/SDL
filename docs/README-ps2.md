PS2
======
SDL port for the Sony Playstation 2 contributed by 
   Francisco Javier Trujillo Mata

Credit to
   Francisco Javier Trujillo Mata - fjtrujy@gmail.com

Building
--------
To build for the PS2, make sure the PS2Dev environment is ready and variable defined in the path and run:
   make -f Makefile.ps2
